# Инструкции для Claude Code

Язык общения с пользователем — русский.

## Стек (использовать всегда)

- Один HTML файл, всё внутри: CSS в `<style>`, JS в `<script>`
- Никаких фреймворков и внешних JS-библиотек кроме Google Fonts
- CSS-переменные в `:root` для цветов, шрифтов, отступов
- Vanilla JS для интерактивности
- Google Fonts — два шрифта: display + body
- Адаптивность обязательна, mobile-first
- Файл должен работать без сборки и npm

## Стандартные переменные (использовать во всех проектах)

Имена полей форм всегда одинаковые:

- `name` — имя
- `phone` — телефон
- `email` — email
- `telegram` — Telegram
- `whatsapp` — WhatsApp
- `instagram` — Instagram
- `message` — сообщение / вопрос
- `form_name` — название формы (заполняется скриптом автоматически)
- `page_url` — URL страницы (заполняется скриптом автоматически)
- `utm_source`, `utm_medium`, `utm_campaign`, `utm_content`, `utm_term`
- `gclid`, `fbclid`, `ttclid`, `fbp`, `fbc`, `ga_client_id`

## Поведение при создании лендинга

1. Создать лендинг по запросу пользователя
2. После создания всегда выводить:

"✅ Лендинг создан.
Следующий шаг — подключить формы и аналитику.
Напишите: /land-agent"
