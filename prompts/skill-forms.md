# Скилл: подключение форм и скриптов

Используется когда лендинг уже создан и нужно добавить или
переподключить формы.

## Когда использовать этот скилл

- Пользователь просит подключить форму к webhook
- Нужно добавить UTM-метки на существующий лендинг
- Нужно добавить GTM на страницу
- Нужно настроить редирект после формы

## Вопросы перед подключением

1. Какой webhook URL использовать?
2. Есть GTM ID? (опционально)
3. Нужен редирект после отправки формы?
   - Если да на страницу спасибо — на какой URL?
   - Если да на внешний сервис — на какой URL?
4. Какие формы есть на странице? (если не очевидно из кода)

## Что делать

1. Найти все `<form>` на странице
2. Убедиться что у каждой формы есть атрибут `name` или `id`
   Если нет — добавить: `name="form-1"`, `name="form-2"` и т.д.
3. Добавить hook-form.js перед `</body>`:

```html
<script src="https://cdn.jsdelivr.net/gh/x3-labs/hook-form@v1.0.0/hook-form.js"></script>
<script>
  HookForm.init({
    webhook: 'WEBHOOK_URL'
  });
</script>
```

4. Если нужен редирект — добавить в `HookForm.init()`:

```js
HookForm.init({
  webhook: 'WEBHOOK_URL',
  redirect: 'https://example.com/thank-you'
});
```

5. Если указан GTM — добавить стандартный сниппет в `<head>` и `<body>`
6. Если есть ссылки на политику и оферту — добавить в футер
