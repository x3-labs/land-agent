# Стандартные названия переменных

Использовать одинаково во всех проектах.

## Поля форм

| Переменная | Описание              |
|------------|-----------------------|
| name       | Имя пользователя      |
| phone      | Номер телефона        |
| email      | Email адрес           |
| message    | Сообщение / вопрос    |

## Служебные поля (заполняются скриптом)

| Переменная | Описание                        |
|------------|---------------------------------|
| form_name  | Название формы                  |
| page_url   | URL страницы без UTM-параметров |

## UTM-метки

| Переменная   | Описание               |
|--------------|------------------------|
| utm_source   | Источник трафика       |
| utm_medium   | Канал трафика          |
| utm_campaign | Название кампании      |
| utm_content  | Содержание объявления  |
| utm_term     | Ключевые слова         |

## Click ID и пиксели

| Переменная     | Описание                   |
|----------------|----------------------------|
| gclid          | Google Ads Click ID        |
| fbclid         | Facebook Click ID          |
| ttclid         | TikTok Click ID            |
| fbp            | Facebook Browser ID        |
| fbc            | Facebook Click ID (кука)   |
| ga_client_id   | Google Analytics Client ID |
